# NYTEMODE Academy

Free AI & Tech Education platform.

## Stack

- **11ty (Eleventy)** - Static site generator
- **Decap CMS** - Content management
- **Netlify** - Hosting
- **Stripe** - Payments (coming soon)

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
